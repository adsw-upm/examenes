Se desea programar una aplicación llamada ParApp, que recibiendo una entrada de texto del usuario indique si su tamaño es par o impar. Para ello, recibimos una interfaz gráfica ya diseñada que incluye:
    - Un EditText con id “etEntrada”
    - Un TextView con id “tvResultado”
    - Un botón con id “bComprobar”

Cuando el usuario pulsa el botón se ha de comprobar el tamaño de la cadena de texto introducida en el campo de entrada de texto. Pueden darse 3 situaciones:
    - Si el tamaño de texto es 0, se ha de lanzar un aviso al usuario para que introduzca una cadena de texto.
    - Si el tamaño es par, el campo de texto de salida se actualiza con el siguiente mensaje: “El texto introducido tiene un tamaño par” seguido del tamaño de la cadena entre paréntesis. Ej: (6).
    - Si el tamaño es impar, el campo de texto de salida se actualiza con el siguiente mensaje: “El texto introducido tiene un tamaño impar” seguido del tamaño de la cadena entre paréntesis. Ej: (5).

- (a) (0,5 puntos) ¿Mediante qué atributo asociaría la ejecución de un método Java cuando el botón es pulsado por el usuario?

??? note "Mostrar solución"
    ```java
    onClick
    ```

- (b) (0,5 puntos)

??? note "Mostrar solución"
    ```java
    public void CompruebaParidad(View V) {

        EditText et = findViewById(R.id.etEntrada);
        TextView tv = findViewById(R.id.tvResultado);

        int len = et.getText().toString().length();

        if (len == 0) {
            Toast.makeText(this, "Introduzca texto", Toast.LENGTH_SHORT).show();
            tv.setText("");
        } 
        else if (len % 2 == 0) {
            tv.setText("Es par (" + len + ")");
        } 
        else {
            tv.setText("Es impar (" + len + ")");
        }
    }
    ```
