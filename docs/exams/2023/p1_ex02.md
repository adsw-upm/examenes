# Ejercicio 2 (2.5 puntos)

Dado el siguiente código:

```java
static void funcion(int a[]) {    
    int start = 0;
    while (start < a.length) {      
        int pos = start;    
        for (int i = start + 1; i < a.length; i++)     {
            if (a[i] > a[start]){
                pos++;
            }
        }

        if (pos == start) {
            start++;
        } else {
            int element = a[start];
            a[start] = a[pos];
            a[pos] = element;
        }
        System.out.println(pos);
    }
}
```

Se pide:

1. (2 puntos) ¿Qué valor tendrá el array `arr` tras ejecutar la instrucción `funcion(arr);` y qué texto se mostrará por consola durante su ejecución en los siguientes casos?
   - Cuando `int[] arr = {1, 4, 3, 2, 5};`
   - Cuando `int[] arr = {1, 2, 3, 2, 1};`
2. (0.5 puntos) Estime el orden de complejidad del algoritmo. Razone su respuesta.

??? note "Mostrar solución"
    1. Salida por consola:
    ```
    4
    0
    1
    2
    3
    4
    ```
    Y `arr = {5, 4, 3, 2, 1}`
    
    
    2. $O(n)$, siendo $n$ el tamaño del array `a`.
