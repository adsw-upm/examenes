Dada la clase `FraseCelebre`, que se utiliza para almacenar información sobre frases célebres, incluyendo el año en que se dijo la frase, el autor de la frase, y la frase en sí. Se debe desarrollar el código necesario para que sus objetos puedan ser ordenados. El criterio de ordenación es, primero se debe ordenar por año de más actual a más antigua y segundo por orden alfabético del nombre del autor.

```java
public class FraseCelebre {

    // Atributos de la clase
    private int anio;        // Año en que se dijo la frase
    private String autor;    // Autor de la frase
    private String frase;    // La frase célebre

    // Constructor de la clase
    public FraseCelebre(int anio, String autor, String frase) {
        this.anio = anio;
        this.autor = autor;
        this.frase = frase;
    }

    // Métodos getters y setters para los atributos
    ...
}
```

Se pide:

- (a) (2 puntos) Modifique, añade o desarrolle el código necesario para que tras la ejecución del siguiente código de ejemplo se muestre por consola las frases célebres ordenadas con el criterio de ordenación indicado. 

??? note "Mostrar solución"
    ```java
    public class FraseCelebre implements Comparable<FraseCelebre> {

        // Atributos de la clase, getters y setters
        ...

        // Método compareTo para comparar objetos FraseCelebre
        @Override
        public int compareTo(FraseCelebre otraFrase) {

            // Primero compara por año
            int comparacionAnio = Integer.compare(this.anio, otraFrase.anio);

            // Se devuelve el valor negado para que el orden sea de más moderna a más antigua
            if (comparacionAnio != 0)
                return -comparacionAnio;

            // Segundo criterio: se compara por el nombre del autor
            return this.autor.compareTo(otraFrase.autor);
        }
    }
    ```

- (b) (2 puntos) Desarrolle el método `public static String calcularModaDeAutores(ArrayList<FraseCelebre> array)`, que recibe como parámetro un `ArrayList` de objetos `FraseCelebre` y calcule la moda de los autores de las frases célebres del `ArrayList`. La moda se define como el valor que más se repite en un conjunto de datos. En caso de que haya más de un autor tenga la máxima frecuencia, el método puede retornar cualquiera de ellos.

```java
public static void main(String[] args) {
```

??? note "Mostrar solución"
    ```java
    // Método para calcular la moda de los autores
    public static String calcularModaDeAutores(ArrayList<FraseCelebre> array) {
    
        Map<String, Integer> frecuenciaAutores = new HashMap<String, Integer>();
        int maxFrecuencia = 0;
        String moda = null;
    
        for (FraseCelebre frase : array) {
            String autor = frase.getAutor();
            int nuevaFrecuencia;
    
            // Verificamos si el mapa ya contiene el autor
            if (frecuenciaAutores.get(autor) != null) {
                // Si el autor ya está en el mapa, incrementamos su frecuencia
                nuevaFrecuencia = frecuenciaAutores.get(autor) + 1;
            } else {
                // Si el autor no está en el mapa, su frecuencia es 1
                nuevaFrecuencia = 1;
            }
    
            frecuenciaAutores.put(autor, nuevaFrecuencia);
    
            // Actualizamos la moda si se supera la frecuencia máxima
            if (nuevaFrecuencia > maxFrecuencia) {
                maxFrecuencia = nuevaFrecuencia;
                moda = autor;
            }
        }
    
        return moda;
    }
    ```
